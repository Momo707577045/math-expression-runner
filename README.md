# 数学运算执行器

在线数学表达式计算器，无按钮设计，输入即算。适合程序员或日常需要快速算式的场景，无需点按计算器按键，直接输入表达式即可得到结果。

## 功能

- **整段计算**：在输入框中输入如 `1 + 2 * 3`，按运算优先级实时得到结果（此处为 7）。支持加减乘除与括号，与常见计算器逻辑一致。
- **选区计算**：用鼠标划选表达式中的某一段（例如在 `1 + 2 * 3` 中只选中 `2 * 3`），则仅对选中部分求值，结果单独显示，方便做局部验算或分步计算。
- **排除数字**：可点击/选中某些数字将其置灰，这些数字在后续计算中会被排除，适用于「暂时不看某几项」的试算场景。

## 使用方式

1. **网页使用**：在本地直接双击打开 `index.html`，或将该目录部署到任意静态服务器后通过浏览器访问。输入内容会写入 localStorage，刷新后仍会保留。
2. **油猴浮窗**：安装同目录下的 `math-expression-runner.user.js`（需先安装 Tampermonkey 等油猴扩展）。安装后在任意网页中，通过油猴菜单选择「开启/关闭」即可在页面右下角唤出小型浮窗，无需离开当前页面即可使用运算器；再次选择可关闭浮窗。

## 技术说明

- 单页应用，基于 Vue 与 contenteditable 实现可编辑表达式区域；从输入中提取数字与运算符（`+ - * / ( )`）后，经简单拼接与规范化，使用 JavaScript 的 `eval` 进行求值。
- 支持整数、小数及括号嵌套；是否显示「规范化后的表达式」可在界面中切换。当前表达式与显示偏好会持久化到 localStorage，便于下次打开时延续使用。
